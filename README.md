# 🎨 Cassourel – Créateur de Carrousels

Transforme ton texte en carrousel visuel prêt à poster sur Instagram.

## ✨ Fonctionnalités

- **Découpe automatique** : chaque ligne commençant par un chiffre devient une slide
- **9 designs** : Minimal, Bold, Floral, Cadre, Pointillés, Diagonal, Dégradé, Lignes, Cercle
- **8 polices** : Playfair, Dancing Script, Bebas Neue, Cormorant, Nunito, Oswald, Space Mono, DM Sans
- **Couleurs personnalisables** : fond + texte
- **3 formats** : Carré 1:1, Portrait 4:5, Stories 9:16
- **Aperçu en temps réel**
- **Téléchargement** slide par slide ou toutes d'un coup

## 🚀 Déploiement sur Netlify

1. Fork ce repo sur GitHub
2. Va sur [netlify.com](https://netlify.com) → **Add new site** → **Import from Git**
3. Sélectionne ce repo
4. Laisse les paramètres par défaut (pas de build command nécessaire)
5. Clique sur **Deploy**

## 📁 Structure

```
cassourel/
└── index.html   # L'application complète (tout-en-un)
```

## 💡 Comment utiliser

1. Colle ton texte dans la zone de gauche
2. Choisis un design, une police, des couleurs
3. Clique sur **Générer les slides**
4. Navigue entre les slides avec les flèches
5. Télécharge chaque slide en PNG

## 📝 Format du texte

```
Titre de ta publication 🌸

Texte d'intro ou sous-titre

1. Premier conseil 🥺
Description du premier conseil...

2. Deuxième conseil 💕
Description du deuxième conseil...

Ton CTA ici – Abonne-toi !
```
